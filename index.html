❌ 데이터 파일 로드 실패

CSV 파일을 불러오지 못했습니다. 아래 원인을 확인해 주세요.

  1. 파일 위치 확인: 서울_기온.csv, 양평_기온.csv, 전력수요.csv 파일이 본 index.html과 동일한 폴더에 있는지 확인하세요.
  2. 로컬 보안 정책(CORS) 제한: 브라우저에서 HTML 파일을 더블 클릭하여 열면 보안상 CSV 파싱이 제한됩니다.
  3. 해결 방법: VS Code의 Live Server 확장 프로그램을 사용해 구동하거나, GitHub Pages에 업로드하여 확인하시면 정상 작동합니다.
